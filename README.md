# Data Science Template
A base template for your basic data science / data analysis project.
This is a simple version of [the cookiecutter template](https://drivendata.github.io/cookiecutter-data-science/).

## Structure

```
    |--data
        |--raw
        |--interim
        |--processed
        |--externals
    |--models
    |--notebooks
    |--reports
        |--figures
    |--src
    |--requirements.txt
```
